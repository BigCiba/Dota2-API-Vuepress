# ProgressBar
进度条
# 属性
属性名|类型|描述
--|--|--
min|Float|最小值，默认0.0
max|Float|最大值，默认1.0
value|Float|值
# JS 属性
属性名|类型|描述
--|--|--
min|Float|最小值，默认0.0
max|Float|最大值，默认1.0
value|Float|值
# 范例
```xml
<ProgressBar id="myid" min="0" max="10000" value="5000"/>
```
