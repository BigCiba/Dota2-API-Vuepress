# GetCooldown
```lua
float GetCooldown(iLevel)
```
# Class
✔️ `Server: CDOTABaseAbility`  
❌ `Client: undefined`  

# Function Description
Get the cooldown duration for this ability at a given level, not the amount of cooldown actually left.
