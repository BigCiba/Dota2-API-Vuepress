# AxisAngleToQuaternion
```lua
Quaternion AxisAngleToQuaternion(Vector_1, float_2)
```
# Class
✔️ `Server: Global`  
✔️ `Client: Global`  

# Function Description
(vector,float) constructs a quaternion representing a rotation by angle around the specified vector axis
